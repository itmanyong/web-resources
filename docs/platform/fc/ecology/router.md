# 路由器

::: tip 说明

-   纯前端控制的细致到模块的页面匹配组装能力
-   前后端分离重要的技术创新点，完全摆脱服务端路由的控制

:::

## TanStack Router <ProjectBadge starts='tanstack/router' version='@tanstack/router-core' />
