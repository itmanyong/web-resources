Solid
#

一个吸取多个框架优点汇聚而成的简单高效 性能卓悦的 JavaScript 界面构建库，无VDOM 完全JSX是它最大的特点，官方文档详尽，配有完整教程，本身生态在积极发展中，是一个值得关注的框架

  • 官方文档 英文&中文
  • 公网仓库 Github | Npm
  • 体验示例 官方示例 | 官方教程

Svelte
#

完全静态编译 无VDOM 真正的响应式，将传统与响应式模式结合的产物，自动作用域 自动样式隔离 细粒度更新等都是它的特点，是一个独树一帜的优秀框架

  • 官方文档 英文 | 中文
  • 公网仓库 Github | Npm
  • 体验示例 官方示例 | 官方教程

Alpine
#

轻量的无需编译 类Vue 传统模式 开发框架，只需通过 script 标签加载 sdk 即可在 HTML 中获得类 Vue 式开发体验

  • 官方文档 英文
  • 公网仓库 Github

Nerv
#

基于虚拟 DOM 技术的类 React16 框架，保持完全一致的 API，尺寸更小，兼容IE8是个亮点，有需求的朋友可以考虑一下

  • 官方文档 英文 | 中文
  • 公网仓库 Github | Npm
