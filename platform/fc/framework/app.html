移动端 APP 框架 #

我要推荐

  • 严重影响原生移动 APP 的开发市场，在性能体验不是那么重要的情况下的首选方案

Fair
#

58自研为Flutter设计的动态化框架，通过Fair Compiler工具对原生Dart源文件的自动转化，使项目获得动态更新Widget Tree和State的能力

  • 官方文档 英文 | 中文
  • 公网仓库 Github

Hummer
#

滴滴出品支持使用原生JS Vue React开发一套代码可以同时支持编译为Android和iOS应用

内置多个定制组件可用、使用原生渲染引擎

  • 官方文档 中文
  • 公网仓库 Github | Npm

Weex
#

阿里巴巴出品支持开发Android iOS WEB平台的框架，支持 Vue Rax 原生语法编写，不过目前发展状况不是很好

  • 官方文档 英文 | 中文
  • 公网仓库 Github | Npm

React-native
#

Facebook出品使用React技术开发Android iOS的框架，算是补上 React 生态的重大一环，强大社区生态，对于 React 热爱者来说是不二之选

  • 官方文档 英文 | 中文
  • 公网仓库 Github | Npm

Ionic
#

跨平台的支持以React Angular Vue方式开发移动端应用的专注于UI UX的工具包，官方维护超过100+的组件并配置了超全的文档

  • 官方文档 英文 | 中文
  • 公网仓库 Github | Npm

Flutter
#

谷歌出品出品使用Dart语言进行开发一套代码支持构建安卓 iOS Web Windows macOS Linux 平台的优秀框架

  • 官方文档 英文 | 中文
  • 公网仓库 Github

Framework
#

免费开源的提供超多具有原生外观的组件的可用于开发移动 桌面 Web的集中性框架，可与Electron NW.js等工具结合构建原生桌面应用，本身支持热门框架进行配合使用，包括:React Vue Svelte

  • 官方文档 英文 | Vue-英文 | React-英文 | Svelte-英文
  • 公网仓库 Github
